iOS Development Project: Wordcraft

Wordcraft is my first iOS app, designed to test and expand your vocabulary. Using a dictionary API, the app lets you verify words as you play. The goal is simple: see how many words you can create and how many points you can earn. You can also save your sessions to track your progress over time.

Wordcraft was built using UIKit and Swift in Xcode.

You can view the source for WordCraft on GitHub: https://github.com/nahfilaali/WordCraft.git

Virtual Reality Project

This VR project, developed using Unity and coded in C# with Visual Studio, was designed specifically for Google Cardboard to ensure affordability and accessibility. The project features a fully interactive world controlled with just a single button. The virtual environment includes a diverse terrain, a park area with carts of fruit, and an animated character that reacts to user interactions by being knocked over upon collision. Movement within the world is managed by turning in the desired direction and pressing the button to advance. Additionally, the project includes an open art gallery, adding a creative dimension to the virtual experience.

Animations were sourced from Mixamo, and the project was built using Unity and C# in Visual Studio.

You can view the source for my VR Project on GitHub: https://github.com/nahfilaali/VRPROJECTUnity.git

You can also watch a more in depth video behind the project on YouTube: https://www.youtube.com/watch?v=tpUaC5YK8bM

Weather Application Project

This was a group effort to develop a Weather App using the OpenWeatherMap API, delivering real-time and forecasted weather data for global locations. Working as a full-stack developer, I primarily focused on front-end features to create a seamless and interactive user experience, allowing users to view current and five-day forecasts and quickly access previously searched cities. This project provided hands-on experience with high-demand applications and strengthened my skills in user-centered design, API integration, and creating responsive, user-friendly interfaces.

This application uses Node.js and Express.js for the backend, MongoDB for data persistence, React.js and CSS for the frontend, and the OpenWeatherMap API to deliver real-time weather data.

You can view the source for this project on GitHub: https://github.com/Tabshir2003/Weather-App
